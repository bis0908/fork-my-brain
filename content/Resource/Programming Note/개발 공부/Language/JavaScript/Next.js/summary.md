### Next.js
1. `Node.js` 기반 SSR 프레임워크
2. 모든 페이지는 서버에서 처음 렌더링 한다.
3. `route`
	1. 폴더 구조를 따른다.
	2. `main` 폴더는 app ('/') 이다.
4. `hydration`
	1. 서버에서 html 자체를 던져주면
	2. 클라이언트는 이걸 받고 js(리액트)를 구동, 페이지를 인터렉티브하게 사용할 수 있다.
	3. 이때 파일 상단에 `'use client'` 라고 명시를 해줘야 함.
	4. server --> html ---> React()
5. `layout`
