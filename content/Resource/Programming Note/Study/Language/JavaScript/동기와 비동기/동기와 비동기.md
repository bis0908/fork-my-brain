1. 동기(synchronous)
    1. 동시에 일어난다는 뜻.
    2. 요청과 결과가 동시에 일어나는 약속.
    3. 간단하고 직관적인 설계 가능
    4. 결과가 주어질 때까지 아무것도 못하고 대기를 해야한다.
2. 비동기(Asynchronous)
    1. 동시에 일어나지 않는다는 뜻.
    2. 요청과 결과가 동시에 일어나지 않을 것이라는 약속.
    3. 요청한 그자리에서 결과가 주어지지 않는다.
    4. 복잡한 설계
    5. 결과가 주어지는데 시간이 걸리더라도 그 시간에 다른 작업을 할 수 있어 자원을 효율적으로 사용할 수 있다.
