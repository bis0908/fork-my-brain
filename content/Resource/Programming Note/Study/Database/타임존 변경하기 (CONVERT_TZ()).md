#### CONVERT_TZ(dt,from_tz,to_tz)

- CONVERT_TZ()는 날짜/시간 값 dt를 from_tz가 지정한 표준 시간대에서 to_tz가 지정한 표준 시간대로 변환하고 결과 값을 반환합니다. 표준 시간대는 5.1.15절. "MySQL 서버 표준 시간대 지원"에 설명된 대로 지정됩니다. 이 함수는 인수가 유효하지 않거나 인자 중 하나라도 NULL인 경우 NULL을 반환합니다.

- 32비트 플랫폼에서 이 함수에 지원되는 값 범위는 TIMESTAMP 타입과 동일합니다
  (범위 정보는 11.2.1절. "날짜 및 시간 데이터 타입 구문" 참조). 64비트 플랫폼(MySQL 8.0.28부터)에서 지원되는 최대 값은 '3001-01-18 23:59:59.999999' UTC입니다.

- 플랫폼이나 MySQL 버전에 관계없이 from_tz에서 UTC로 변환할 때 값이 지원되는 범위를 벗어나는 경우 변환이 수행되지 않습니다.

```SQL
mysql> SELECT CONVERT_TZ('2004-01-01 12:00:00','GMT','MET'); 
	-> '2004-01-01 13:00:00' 
mysql> SELECT CONVERT_TZ('2004-01-01 12:00:00','+00:00','+10:00'); 
	-> '2004-01-01 22:00:00'
```
